## 🚀 How to Run Locally

If you want to view the code or run a copy:

```bash
# Clone the repository
git clone https://github.com/SHARIARxSAKIL/Portfolio-skill-2.git

# Go into the project folder
cd Portfolio-skill-2

# Install dependencies
yarn install

# Start the development server
yarn start
